# Church of Our Savior MCC Website

This repository contains the static website for Church of Our Savior MCC, an open and affirming Christian church in Boynton Beach, Florida. Pages are written in vanilla HTML and styled with TailwindCSS via the CDN along with a small custom stylesheet in `assets/css/main.css` and JavaScript helpers in `assets/js/main.js`.

## Getting Started

Open any of the HTML files in your browser or use a simple HTTP server for local development. For example:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000/index.html` to preview the site.

## Key Pages

- `index.html` – Home page with service information and featured ministries.
- `about.html` – Story, values, and MCC affiliation.
- `mission.html` – Worship &amp; ministries overview.
- `beliefs.html` – Stained glass gallery and stories.
- `sermons.html` – Messages, livestream archive, and resources.
- `serve.html` – Giving information.
- `visit.html` – Plan your visit and contact form.

## Assets

Custom styles and scripts live under `assets/`:

- `assets/css/main.css` – Additional styling helpers.
- `assets/js/main.js` – Navigation toggles and utilities.
